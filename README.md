Cannon Fodder
=============

A clone of the 1989 Mac classic Cannon Fodder, a game where two cannons wage war.

Original game by John Barnett, Fantasy Computing, 1989.

Clone by Jordan Dee McConnell, 2014. Clone written in C++ using SDL2.
